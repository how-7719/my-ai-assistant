# 🚀 我的人工智慧助手 v1.0

這是一個專為提升學習效率與生活管理設計的個人助理應用程式，基於 Google Gemini 模型開發，並特別針對**航空管理**學業與 **CTF（網路安全）** 實務進行了優化。

## 🌟 核心功能
* **智能對話：** 採用 Gemini 1.5 Flash 模型，提供極速的語意理解與回覆。
* **上下文記憶：** 具備對話歷史紀錄功能，讓助理更了解使用者需求。
* **跨平台支援：** 透過 PWA 技術，可在電腦瀏覽器與手機主畫面完美運行。
* **多模態分析：** 支援圖片輸入，可分析航空法規講義照片或 CTF 程式碼截圖。

## 🛠️ 技術棧 (Tech Stack)
* **前端框架:** Vite + Vanilla JavaScript
* **AI 引擎:** Google Generative AI SDK (Gemini API)
* **部署平台:** GitHub Pages
* **樣式設計:** CSS3 (Responsive Design)
